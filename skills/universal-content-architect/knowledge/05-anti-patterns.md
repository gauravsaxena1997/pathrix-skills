# Anti-Patterns

The content that reliably underperforms or flags algorithms. Delete these on sight during drafts or review.

## Generic traps (universal)

- **Generic guru takes** : "Here's 5 tips to be more productive." No personal stake, no proof, no specificity.
- **Fluff openers** : "Hey everyone", "Have you ever wondered", "In today's world".
- **List-heavy with no context** : one-line bullets, no stories, no hook.
- **Engagement bait** : "Tag someone who..." or "Like if you agree". Platforms actively deprioritize this.
- **All value, no CTA** : informative but dead-ends. Every post needs a next step.
- **Unclear promise** : hook doesn't state the benefit clearly enough for a stranger to care.
- **Motivational slop** : generic "keep going" content with no lived experience.

## Platform-specific traps

### Instagram

- Reposting TikTok videos with TikTok watermark : triggers shadow-ban signal.
- Topic ping-pong : jumping between unrelated topics destroys topic-cluster signal.
- Ignoring 30-minute engagement window : post and disappear = lost reach.
- Generic high-volume hashtags (#love, #instagood, #photooftheday) : algorithmic graveyard.
- Bot-ish caption tone : algorithm deprioritizes content that doesn't read as human.
- Auto-generated captions with no edit pass.

### Reddit

- Heavy emoji use : Reddit culture rejects it.
- Clickbait titles : use "click-gap" instead. "How I built a $10k/mo agency" > "YOU WON'T BELIEVE...".
- Pitching before proving value : auto-downvoted.
- Cross-posting identical content across multiple subreddits : flagged as spam.
- Ignoring subreddit-specific vocabulary : reads as outsider, tanks engagement.
- Self-promotion ratio > 10% : moderators remove.

### X / Twitter

- Threads without momentum : each tweet must make you want the next.
- No hook at thread start.
- Rewriting the same take repeatedly : audience notices.
- Over-tagging influencers : reads as attention-begging.
- Stale takes : X rewards speed on news; slow takes get lost.

### YouTube

- Cold open with no promise : viewers bounce at 15 seconds.
- Outro that begs for subscribes instead of teasing the next video.
- Thumbnails with more than 4 words or tiny faces.
- Long intros with channel history.

### LinkedIn (when eventually in scope)

- Corporate-polish tone : feels like a press release.
- Humble-brag openers.
- Stock photo headers.
- Over-hashtagged posts (>5).

### Threads

- Posting pure X threads : audience expects different format.
- Ignoring visual culture (Threads users came from IG).

## Friction red flags (pre-publish checklist)

- [ ] More than 1 exclamation mark per 3 sentences?
- [ ] Any paragraph longer than 3 lines?
- [ ] Any banned word from `04-voice-and-tone.md`?
- [ ] Passive voice anywhere?
- [ ] Unclear or missing CTA?
- [ ] Hook doesn't promise a specific outcome?
- [ ] Sounds corporate or over-polished?
- [ ] Em-dash anywhere? (hard no)
- [ ] AI-obvious phrasing?

Any yes = rewrite.

## Algorithm-damage patterns (don't do twice)

- Deleting and re-posting : tanks reach on second post.
- Editing caption heavily after 10 minutes : resets engagement velocity.
- Posting then going offline : misses 30-minute window.
- Sudden niche pivot : confuses topic-cluster ranking.
- Posting 5 things in one day after a week of silence : burst kills ranking.
