# Threads

Meta's X competitor. Audience is mostly IG overflow : younger, more visual, less political, more lifestyle-flavored than X.

## How Threads differs from X

| Dimension | X | Threads |
|---|---|---|
| Audience | Power users, news, politics | IG audience, younger, lifestyle |
| Tone | Sharp, opinionated, reply-driven | Casual, observational, softer |
| Format | Threads + replies | Single posts dominate; threads secondary |
| Virality source | Contrarian takes + reply game | Trending audio, visual culture, reposts |
| Hashtags | Outdated | Still usable, less stigma |

## What works on Threads

- **Casual observations** : "Just realized X. Does anyone else do this?"
- **Aesthetic / lifestyle + opinion**
- **Replies to IG-adjacent creators**
- **Trending hashtag participation**
- **Photo-first posts** (Threads rewards images more than X does)
- **Personal takes without the stakes** : lower-pressure tone
- **Build-in-public but softer** than X's "hustle" flavor

## What doesn't work on Threads

- Pure text walls (X-style threads without visuals)
- Corporate or polished tone
- Hard contrarian takes that start fights (X audience wants this; Threads audience bounces)
- Heavy self-promotion
- Newsjacking political hot takes

## Posting cadence

Low-stakes. Treat Threads as an IG bleed audience : same voice, shorter format, less polish.

Opt-in per post from Pathrix Dispatch. Default Threads toggle off. Enable when content fits (casual observations, visual posts, lifestyle takes).

## Hook patterns that work

- Observation: "Nobody talks about [thing] but..."
- Quick take: "Hot take: [X]"
- Question: "Is it just me or does [Y]?"
- Micro-story: one-paragraph anecdote
- Visual + caption combo

## Format guidelines

- One idea per post
- Keep under 300 characters for punch
- Images outperform text-only
- Reply game matters (same as X)
- Reposting (Threads equivalent of retweet) is a distribution vector

## Repurposing from other platforms

- **X thread** : lead tweet becomes a Threads post; don't paste the whole thread
- **IG caption** : first 2 sentences become a Threads post
- **Reddit post** : the one-liner lesson becomes a Threads observation
- **YouTube Short** : Threads post with a snippet of the idea + link to Short

## Integration note (Dispatch)

Threads uses the Meta Graph API family : same auth as Instagram. Bundling Threads with IG in Dispatch is cheap (hours, not days).

Default: Threads toggle OFF per post. Turn on when content is Threads-appropriate. No dedicated pillar strategy. Revisit in 90 days of usage data.

## Anti-patterns

- Pasting X posts verbatim
- Heavy marketing tone
- Over-hashtagging
- Political hot takes
- Treating Threads as a secondary channel with zero care (bad posts hurt IG reach too, since the accounts are linked)
